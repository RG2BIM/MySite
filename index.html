Плагины Revit (pyRevit)

Плагины для облегчения работы специалистов при работе с цифровыми информационными моделями в среде AutoDesk Revit 2023 и выше.
Плагины основаны на бесплатный открытый фреймворк pyRevit, предназначенный для быстрой автоматизации и создания собственных инструментов с использованием языка Python.

ScriptManagerPlugin (SMP)

Специальный плагин-менеджер для Renga, который позволяет создавать собственные плагины минуя все промежуточные шаги связанные с компиляции файлов DLL, настроек графического интерфейса в коде и бесконечные перезапуском программы при тестировании.
Вы просто создаете папки и кладете туда свои скрипты C# сохраненные в обычном блокноте.
